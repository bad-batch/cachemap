Optional _inWorker

_inWorker: boolean

This private property is set within the WorkerCachemap to tell the instance of the DefaultCachemap it is being used within a web worker.

Optional disableCacheInvalidation

disableCacheInvalidation: boolean
  • Defined in types.d.ts:31

This property disables all cache invalidation. Stale entries will not be culled from the Cachemap instance and the reaper will not run at set intervals if this property is set to true.

Optional indexedDBOptions

indexedDBOptions: IndexedDBOptions
  • Defined in types.d.ts:39

This property specifies the database name and object store name to be provided to IndexedDB. If none are provided, the database name defaults to "keyval-store" and the object store name defaults to "keyval".

Optional maxHeapSize

maxHeapSize: object
  • Defined in types.d.ts:47

This property specifies the approximate maximum memory the Cachemap instance can use on the client and server. IndexedDB and LocalStorage default to 5MB, Map defaults to 1MB, and Redis defaults to Infinity.

Optional mockRedis

mockRedis: boolean
  • Defined in types.d.ts:53

If true, this property replaces Redis with a mocked version of the library, which is useful for testing.

name

name: string
  • Defined in types.d.ts:60

The name of the Cachemap instance, which is used to prefix the key backed up metadata is stored against, as well as prefix all LocalStorage data keys.

Optional reaperOptions

reaperOptions: ReaperOptions

The property specifies configuration options passed to the reaper: the interval time and whether to start the reaper on initialisation.

redisOptions: ClientOpts
