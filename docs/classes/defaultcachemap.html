Class DefaultCachemap

An isomorphic cache that works on the server and the browser, that can use Redis, LocalStorage, IndexedDB or an in-memory Map.

import { DefaultCachemap } from "cachemap";

const cachemap = await DefaultCachemap.create({
  name: "alfa",
  use: { client: "localStorage", server: "redis" },
});
